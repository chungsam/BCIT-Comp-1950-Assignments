Two Places You Must Visit!

  1. 1. Tofino, BC

    Nestled in the western edge of Vancouver Island, Tofino is a truly otherworldly place that is less than a day's commute away. The beaches are a beginner surfer's paradise, with winds just strong enough and waves perfect for both first-time and experience surfers.

    You can also hike along the coast and take in the views of the coast and listen to waves crashing against the rocks that edge the innumerable crevices that line the coast.

    Go out and explore!

  2. 2.Cultus Lake, BC

    Less than an hour's drive from Vancouver, Cultus Lake is a beautiful park that is perfect for a picnic/barbecue, swimming, boating, and other water activities. The lake is vast that you'll mistaken it for the ocean.

    Once you've spent some time at the beach, you can have some fun at Cultus Lake Waterpark, one of the biggest waterparks in Western Canada. And if you still feel restless, you can go visit some nearby wineries, as well as hike or mountain bike on the many trails that are just a few minutes' drive away.

    Get out there soon!

Two New HTML5 Elements

The below content was taken from the Mozilla Developer's Network official HTML element reference pages.

Element Semantics/Rules of Use Attributes Browser Compatibility Notes
<progress>

The HTML <progress> element represents the completion progress of a task, typically displayed as a progress bar.

max

This attribute describes how much work the task indicated by the progress element requires. The max attribute, if present, must have a value greater than zero and be a valid floating point number. The default value is 1.

value

This attribute specifies how much of the task that has been completed. It must be a valid floating point number between 0 and max, or between 0 and 1 if max is omitted. If there is no value attribute, the progress bar is indeterminate; this indicates that an activity is ongoing with no indication of how long it is expected to take.

  • Chrome: Yes
  • Edge: Yes
  • Firefox :Yes
  • IE: 9.0 and above
  • Opera: 11.0 and above
  • Safari: Yes

<mark>

The HTML <mark> element represents highlighted text, i.e., a run of text marked for reference purpose, due to its relevance in a particular context. For example it can be used in a page showing search results to highlight every instance of the searched-for word.

Only global attributes.
  • Chrome: Yes
  • Edge: Yes
  • Firefox :Yes
  • IE: 9.0 and above
  • Opera: 11.0 and above
